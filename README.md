1. vue, node, es 등등 개발 스터디
- 하고 싶은것들 다 해보자.
- front 단은 vue 로 개발하고
- back 단은 node로 개발하고
- 통계 등등의 작업은 es 해보고
- 타 조직에서 무언가 하고 있는것들 나도 적용해 

2. 그날그날 운동하는것을 쉽게 기록 하도록 한다.
- 전에 운동했던 포맷을 그대로 가져와서 그날 운동한 데이터만 입력할 수 있다.
- 새롭게 운동한 기록을 등록할 수 있다.
- 운동한것을 수정할 수 있다.

3. 내가 운동을 잘 하고 있는지 체크를 쉽도록 한다.
- 전에 했던 운동기록과 비교하여 증감 상태를 쉽게 파악 할 수 있다.
